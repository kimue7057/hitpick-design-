# Hitpick Design Mockups

This repository stores portable HTML mockups for the Hitpick design direction.

## Included mockups

- `mockups/home-v2-existing-tone.html`
  - Main page mockup
  - Keeps the existing Hitpick tone while shifting the message toward content IP and monetization
- `mockups/project-detail-v1-existing-tone.html`
  - Project detail top-section mockup
  - Keeps the current participation structure while improving information hierarchy

## How to use

1. Open `index.html` in a browser for a simple entry page.
2. Open each file in `mockups/` directly if you want to inspect only one screen.
3. Continue iterating in plain HTML/CSS first, then move the selected direction into the real product codebase.

## Notes

- These repo versions are self-contained so they can travel easily across machines.
- Image assets are embedded in lightweight portable form inside the HTML files.
- The higher-fidelity local working files still exist in the original Codex workspace.
